📂 TEMPLATE VIDEOS FOLDER

Folder này chứa các template videos có sẵn để user swap face vào.

🎬 CÁCH SỬ DỤNG:
1. Thêm file video (MP4, AVI, MOV, WEBM) vào folder này
2. Video sẽ tự động xuất hiện trong danh sách templates
3. User chỉ cần upload ảnh khuôn mặt để swap vào template

💡 ĐỀ XUẤT TEMPLATES:
- Dance videos
- Funny videos  
- Meme videos
- Action scenes
- Promotional videos
- Tutorial videos

⚡ PROVIDERS SUPPORT:
- Replicate: ✅ Support (stable, $0.14/video)
- VModel: ✅ Support (faster, ~$0.10/video)
- Auto mode: ✅ Replicate primary → VModel fallback

🎯 USE CASE:
User không cần upload video riêng, chỉ cần:
1. Chọn template có sẵn
2. Upload face image
3. Click swap → Done!

📌 NOTE:
- Audio trong template sẽ được preserve
- Template videos nên ngắn (10-60s) để xử lý nhanh
- Chất lượng tốt (720p+) cho kết quả tốt nhất
